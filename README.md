# Netflix Data Analysis

## Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix dataset to understand content trends, ratings, countries, and release patterns.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset
Netflix Movies and TV Shows dataset containing information about:
- Movies
- TV Shows
- Directors
- Cast
- Countries
- Ratings
- Release dates

## Analysis Performed
- Movies vs TV Shows comparison
- Content rating distribution
- Movie duration analysis
- Release year trends
- Top countries producing Netflix content

## Key Insights
- Netflix has more movies than TV shows.
- The USA produces the highest number of titles.
- Netflix content increased significantly after 2015.
